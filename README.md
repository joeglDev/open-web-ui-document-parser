# Document extraction using Open Web UI

Uses Open Web UI to extract information from a PDF file and output as a json file.

## Quick start

- Run `touch .env` to create an environment file to store your secrets. Copy secrets from '.env.example' file
- Run `poetry install` in cli to install dependencies.
- Run the file `main.py`


## Lint and format

The first time run `pre-commit install` in cli to configure the git hooks.
To format and lint using ruff run `pre-commit run --all-files` in cli.
